Gray Code~用十进制表示二进制问题

发表于 2020-02-22

89. gray code
The gray code is a binary numeral system where two successive values differ in only one bit.
Given a non-negative integer n representing the total number of bits in the code, print the sequence of gray code. A gray code sequence must begin with 0.
Example 1:
Input: 2
Output: [0,1,3,2]
Explanation:
00 - 0
01 - 1
11 - 3
10 - 2

分析格雷码的构成的规律：
· 每向左增加一位，格雷码行数变为之前的2倍
· 前半部分前加0，后半部分前加1
· 前半部分0之后保持之前的（正序），后半部分为之前的倒序

找到了规律之后满脑子都是字符串二进制十进制的转换怎么实现的问题，看到别人的博客才意识到完全可以只在十进制上实现，反正最后是返回十进制数。
对于十进制数，前加0操作相当于不变，前加1操作相当于加2的当前位数次方。也就是说前半部分已加入的不需要进行任何操作，只需要在list中加入后半部分。
