Jump Game
Given an array of non-negative integers, you are initially positioned at the first index of the array.
Each element in the array represents your maximum jump length at that position.
Determine if you are able to reach the last index.

DP问题，比较喜欢用top-down解决。
top-down和bottom-up都需要维持一个数据结构记录子问题的结果。

第一个答案存在的问题：

  1. 没有区分unvisited和false，导致false时无法直接利用子问题的解。可以把canJumpFrom存储的值设为true/false/unvisited.
  2. canJumpFrom数组存储了结果，canJump没有必要再每次return结果

其他方法

Greedy

canReachLast = last index，从后last index开始往前，找到第一个能跳到canReachLast的，设为新的canReachLast（每次能跳到canReachLast的也一定能跳到最后），直到canReachLast==0
Time complexity : O(n)
Space complexity : O(1)
（草，这个方法好厉害）
