Minimum Window Substring
Given a string S and a string T, find the minimum window in S which will contain all the characters in T in complexity O(n).
Example:
Input: S = “ADOBECODEBANC”, T = “ABC”
Output: “BANC”

在S中找到一个最短的substring，能包含T中所有的元素（T中的元素可能有重复，T中出现了几次，substring中也要至少出现几次）
先想到一种brute force的方法，从左往右遍历S（记为left），在从left往后走到right，每次判断left到right是否包含所有T中元素。这种方法的时间复杂度是O(n²)，不满足题目要求。

观察上述方法中不必要的操作，可以发现有些left和right的位置一定不会是最短substring的端点，即不属于T的元素所在的位置。比如example中，有可能是minimum substring的只有：”ADOBEC”, “BECODEBA”, “CODEBA”, “BANC”。

为了更快地定位到那些有可能的substring，可以使用sliding window方法：维持一个记录当前substring中T中元素个数的容器，right前进直到满足T条件，left前进直到刚刚不满足T条件，此时得到的substring再和当前最小比较。left和right都只走了一遍S，因此时间复杂度是O(n)。
