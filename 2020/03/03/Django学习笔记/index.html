Django初体验

发表于 2020-03-03 更新于 2020-04-18 分类于 Django

Create first Django project

install

$ pip install django==2.1

create django project

$ django-admin startproject pyshop .
django-admin: a command line utility
startproject: argument
pyshop: argument(project name)
.: in current folder

package structure
pyshop
· __init__.py
· settings.py
· urls.py
· wsgi.py - web server gateway
manage.py - start webserver/database…

run project

python3 manage.py runserver

create app

One Django project can contain multiple apps. Apps are small functional areas in a project.

$ python3 manage.py startapp products # create a new app name’s products
package structure
products
· migrations
· __init__.py
· admin.py # define admin penal
· apps.py # app config
· models.py # define classes
· tests.py # for auto-testing
· views.py # define pages

create view

  1. create views function

  2. create products url mapping

  3. add app(product) url mapping to project

create model

create db automatically

  1. add ‘products.apps.ProductsConfig’(config path) to settings.INSTALLED_APPS
  2. create migration
    $ python3 manage.py makemigrations
    Migrations for ‘products’:
    products/migrations/0001_initial.py
    • Create model Product
  3. run the new migration to genarate db table
    $ python3 manage.py migrate

    每次新建或修改模型之后，都要创建一个新的migration，然后运行。

admin

Django application have a admin panel installed by default.

create super user

$ python3 manage.py createsuperuser
http://127.0.0.1:8000/admin/

register the model we need to manage

P: Got OperationalError when I add product using admin panel.
S: Upgrade Django to v2.2:
$ pip install -U django==2.2
then remake migrations and run it.

customize the admin

add more model to admin panel

templates

add templates

  1. create templates/index.html
  2. render html in views.py

use bootstrap

  1. create base.html. copy Starter template from getbootstrap.com to base.html

  2. set the body content

  3. extend base.html template

为了让其他apps也能继承base.html，最好在products app外建一个templates folder，放base.html
